
### 方式二：生产模式

```powershell

```

**2. 查看服务状态**
```powershell
tasklist | findstr node
```

**3. 停止服务**
```powershell
taskkill /F /IM node.exe
```

## 五、配置说明

### 修改端口

编辑 `server/index.js`，找到：
```javascript
app.listen(3001, ...)
```

将 `3001` 改为其他端口（如 80、8080）

### 修改数据文件位置

编辑 `server/index.js`，找到：
```javascript
const store = new JsonStore('./data/q12survey.json')
```

### 启用外网访问

编辑 `server/index.js`，找到：
```javascript
app.listen(3001, ...)
```

改为：
```javascript
app.listen(3001, '0.0.0.0', ...)
```

这样其他电脑就能通过 `http://服务器IP:3001` 访问

## 七、常见问题

### Q: 如何让其他同事访问？
**A:** 
1. 确保服务器电脑的防火墙允许 3001 端口
2. 其他同事访问 `http://服务器电脑IP:3001/admin`

### Q: 如何修改管理员密码？
**A:** 登录后台 → 用户管理 → 点击用户右侧"重置密码"

### Q: 服务启动失败怎么办？
**A:** 
1. 检查端口是否被占用：`netstat -ano | findstr :3001`
2. 杀掉占用进程后重试

### Q: 如何迁移到新电脑？
**A:** 
1. 复制整个项目文件夹到新电脑
2. 安装 Node.js（版本 18+）
3. 运行 `npm install`
4. 复制 data 文件夹到新电脑

## 八、生产环境建议

1. **使用 PM2 管理进程**
```powershell
npm install -g pm2
pm2 start server/index.js --name q12-survey
pm2 save
pm2 startup
```

2. **配置 Nginx 反向代理**（可选）
```nginx
server {
    listen 80;
    server_name q12.yourcompany.com;
    
    location / {
        proxy_pass http://127.0.0.1:3001;
    }
}
```

3. **定期备份数据文件**
