# Q12组织氛围诊断调研系统

企业内部Q12组织氛围诊断调研平台，支持匿名问卷收集、数据可视化看板和AI智能分析。

---

## 🚀 快速开始

### 第一次使用

1. **双击运行 `安装依赖.bat`**（只需运行一次）
2. **双击运行 `启动系统.bat`**
3. 浏览器自动打开后台管理页面

### 后续使用

直接双击 **`启动系统.bat`** 即可

---

## 📋 功能说明

| 功能 | 说明 |
|------|------|
| 问卷收集 | 员工匿名填写，1-10分量表评分 |
| 链接分发 | 按事业部/部门生成独立问卷链接 |
| 数据看板 | 雷达图、柱状图实时展示得分 |
| AI分析 | 自动生成诊断报告和改进建议 |
| 权限管理 | 超级管理员可分配事业部管理员 |

---

## 🔐 默认账号

```
超级管理员: admin / admin123
```

---

## 🌐 访问地址

- **管理后台**: http://localhost:3000/admin
- **问卷链接**: http://localhost:3000/survey/{token}

---

## 📁 文件说明

| 文件 | 用途 |
|------|------|
| 启动系统.bat | 一键启动前后端服务 |
| 停止系统.bat | 停止所有服务 |
| 安装依赖.bat | 安装Node.js依赖（仅首次需要） |
| data/ | 数据存储目录（自动创建） |

---

## 📊 数据说明

- 数据文件位置: `data/q12survey.json`
- **重要**: 定期备份此文件！

---

## ❓ 常见问题

**Q: 启动后浏览器没反应？**
A: 手动打开浏览器访问 http://localhost:3000/admin

**Q: 提示端口被占用？**
A: 先运行 `停止系统.bat`，再重新启动

**Q: 如何重置管理员密码？**
A: 删除 `data/q12survey.json` 文件，然后重启系统
